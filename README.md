# 🛒 Shopzy - E-commerce Application

A scalable E-commerce Cart application built with React.js and Redux Toolkit, featuring dynamic cart management and responsive UI.

---

## 🚀 Features

- Add to cart & remove items
- Dynamic total price calculation
- Centralized state management using Redux Toolkit
- Responsive product listing UI

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- JavaScript (ES6+)
- Bootstrap
- Vite
